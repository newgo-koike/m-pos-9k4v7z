🎯 1. このツールが目指すもの

このツールは 「世界一の行動管理ツール」 を目指して作られています。
最終ゴールから順番に「要素・マイルストーン・ギャップ・コミットメント・アクション・数字」までを一気通貫で見える化し、
下のレイヤーで数字を入れたら、上の進捗まで自動で動く体験を提供します。

🎯 ゴール
  └─ 🧩 要素（達成に必要な構成要素）
       └─ 📍 マイルストーン（3ヶ月単位）
            └─ 📐 ギャップ（現状とのズレ）
                 └─ 🤝 コミットメント（こうなる！宣言）
                      └─ ⚡ アクション（週ごと）
                           └─ 📊 数字（測定）
                                ↑
                       下から上へ進捗がロールアップ

🚀 2. 最初の30分で立ち上げる

  1. ゴールを書く：ヘッダーの「あなたのゴール…」をクリック → 1年後の到達点を1行で書く（例：2026年12月までに月商300万を3ヶ月連続で達成する）
  2. 要素を3〜5個に分解：行動設計タブの「設計」ビューで「＋ 要素を追加」。ゴールを構成する大きな柱を入れる（例：見込み客10名／成約率50%／紹介ルート構築）
  3. マイルストーン：各要素に対して「3ヶ月後の到達点」を1〜2個
  4. ギャップ：現状とマイルストーンの差を数字で（例：+4名、+7件、+50pt）
  5. コミットメント：そのギャップを埋めるために「こうする！」と決める（例：毎月10件の紹介依頼面談を実施）
  6. アクション：今週やる具体的な動き3〜5個に落とす

💡 ヒント：最初は粗くてOK。毎週の振り返りで微調整していく前提です。

📅 3. 毎週・毎月のルーティン

頻度所要やること
毎日1〜3分⚡今週タブで進捗チェック・数字入力・チェックボックス
週末（金 or 日）15分① 数字を確定 → ② 🔄振り返りタブでKPT記入 → ③ 滞留アクション処理 → ④ 翌週のアクション3〜5件設計

3ヶ月90分🎯 行動設計を全体見直し（要素・マイルストーン・コミットを再設計）

📊 4. KPIタブの「決定 / 見込」の使い分け

各週セルは 上段=決定（確定済み） / 下段=見込（パイプライン・予定） の2値構造。

  • 決定：すでに獲得した成果（成約済み売上、完了したアポ）
  • 見込：これから入る予定（クロージング待ちの案件、予約済みアポ）
  • 決定計：確定累計／見込計：予定累計／着地%：(決定+見込)÷目標 で計算

💡 セクション名（主要指標 / 内訳 / 行動量）はクリックで自由に書き換え可能。業種に合わせてカスタマイズしてください。

🎨 5. テーマ切替・データのバックアップ

  • テーマ切替：ヘッダー右の ⌂ ⚔ ◆ で3世界観を切替（Default/Cockpit/Sci-Fi）
  • JSON Export/Import：⚙ 設定タブからバックアップ可能。別端末への移行もこれでOK
  • テンプレート読込：5プリセット（営業／クリエイター／経営者／個人目標／白紙）から選んでスタート

🤝 6. つまずいたら

  • カスケードを書けない→ 1個だけでもOK。完璧を目指さない
  • 数字が決められない→ 「測れる数字に置き換える」を1on1で兵頭さんと相談
  • 滞留アクションだらけ→ そもそも今週のキャパを超えてる証拠。3個に絞る
  • 毎週続かない→ 金曜午後 or 日曜夜に「15分ブロック」をカレンダーに固定
